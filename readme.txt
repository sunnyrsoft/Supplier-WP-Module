Manage suppliers, categories, and products with custom fields in WordPress.

== Description ==

A comprehensive plugin to manage:
- Supplier Categories with custom fields
- Suppliers with multiple addresses
- Products connected to suppliers

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/supplier-product-manager` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the new menu items to create categories, suppliers, and products
